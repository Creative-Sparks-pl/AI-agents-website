Architecture & Workflow

6 etapów precyzyjnego procesu.

Każdy krok kończy się deliverablem. Ty decydujesz, agent wykonuje.

01

Discovery

Opisujesz swój projekt i cele w oknie czatu. Agent ekstrahuje kluczowe dane.

OUTPUT: project_context.md
02

Strategia i IA

Agent proponuje format strony, hierarchię informacji i mapę serwisu.

OUTPUT: ux_strategy.md
03

Inventory

Dla dużych projektów tworzy listę unikalnych szablonów i komponentów.

OUTPUT: template_inventory.xlsx
04

Wireframing

Agent proponuje sekwencję sekcji z gotowym, merytorycznym copy.

OUTPUT: wireframe.md
05

UX Audit

Agent audytuje dowolny deliverable pod kątem zasad użyteczności i heurystyk.

OUTPUT: audit_report.md
06

Decision Log

Każda ważna decyzja projektowa jest zapisywana wraz z uzasadnieniem.

OUTPUT: decision_log.md
pause

Na każdym etapie agent zatrzymuje się i pokazuje opcje. To Ty jesteś architektem. Agent jest Twoim narzędziem.
